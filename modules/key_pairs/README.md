# Key Pairs Module

Creates Key Pairs on AWS

## Inputs

| Name | Description | Type | Default | Required |
|------|-------------|------|---------|:--------:|
| key_pairs | List of Key_Pairs to create | n/a | yes |

## Outputs

| Name | Description |
|------|-------------|
| private_keys  | List of all private keys |
| public_keys  | List of all public keys |
